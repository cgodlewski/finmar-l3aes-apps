## Mise en ligne recommandee

1. Creer un depot GitHub public dedie a la diffusion etudiante.
2. Pousser au minimum le dossier `docs/`.
3. Dans GitHub :
   `Settings > Pages > Build and deployment > Source = Deploy from a branch`
4. Choisir la branche principale et le dossier `docs`.
5. Recuperer l'URL GitHub Pages et la deposer dans Moodle.

## Structure minimale conseillee

- `docs/index.html`
- `docs/.nojekyll`
- ce `README.md` si vous voulez garder une trace du mode de publication

## Ce que je ne recommande pas de publier

- les scripts de mise a jour
- `update_data`
- les notebooks
- les mappings provider
- la version enseignant complete
