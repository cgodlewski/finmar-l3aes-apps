# Application interactive - Finance de marché L3 AES

Mini-site statique pour publier l'application HTML du cours.

Réalisé par C.J. Godlewski assisté par IA-LLM.

## Avertissement

Cette application est diffusée à des fins exclusivement pédagogiques dans le cadre du cours de Finance de marché en L3 AES.

Les données affichées constituent des instantanés indicatifs de marché. Elles peuvent être incomplètes, approximatives ou ne pas être à jour. Elles ne constituent ni une information financière en temps réel, ni un conseil financier, ni une recommandation d'investissement.
